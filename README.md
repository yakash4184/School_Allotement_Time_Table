# School_Allotement_Time_Table

Frontend-only React application for school timetable management.

## Features

- Upload `.xlsx` timetable files directly in the browser
- Parse Excel data using SheetJS
- Filter timetable by day and class
- Select an absent teacher and view all assigned periods
- Manually assign substitute teachers
- Export the updated timetable back to Excel
- Responsive school ERP style UI with dark mode

## Expected Excel Columns

- Class
- Day
- Period
- Teacher
- Subject

## Run Locally

```bash
npm install
npm start
```

## Build

```bash
npm run build
```
